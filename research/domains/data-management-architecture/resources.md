## 工具资源

### 主流数据库

| 工具 | 类型 | 说明 |
|------|------|------|
| MySQL | 关系型数据库 | 最流行的开源数据库 |
| PostgreSQL | 关系型数据库 | 功能最强大的开源数据库 |
| ClickHouse | 列式数据库 | OLAP 场景高性能 |
| Apache Doris | MPP 数据库 | 国产高性能分析型数据库 |

### 数仓产品

| 工具 | 类型 | 说明 |
|------|------|------|
| Apache Hive | 数据仓库 | Hadoop 生态传统数仓 |
| Apache Spark SQL | 数据仓库 | 内存计算引擎 |
| Apache Flink | 流批一体 | 实时计算框架 |
| SelectDB / Doris | 云原生数仓 | 国产云原生分析型数据库 |

### 数据湖技术

| 工具 | 类型 | 说明 |
|------|------|------|
| Apache Iceberg | 开放表格式 | Netflix 开源，支持 ACID |
| Delta Lake | 开放表格式 | Databricks 开源 |
| Apache Hudi | 开放表格式 | Uber 开源，支持增量处理 |
| Apache Paimon | 湖格式 | Flink 官方项目 |
