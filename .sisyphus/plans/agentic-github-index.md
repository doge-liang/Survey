## Usage

```bash
# Basic mode (no API key) - generates simple table
bun scripts/update-github-index.ts

# AI mode (with KIMI_API_KEY) - generates enhanced README with categories
KIMI_API_KEY="your-key" bun scripts/update-github-index.ts

# Analyze single project (dry run)
bun scripts/analyzer.ts --project github/llama.cpp --dry-run

# Force re-analyze
bun scripts/analyzer.ts --project github/llama.cpp --force
```
